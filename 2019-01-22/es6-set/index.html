学习set

set是ES6中的新增的数据结构，是一种构造函数，类似于数组，但是里面的元素都是唯一的，不重复的。

set的方法

set的操作方法

  • add(val); 添加某个值
  • delete(val); // 删除某个值，返回布尔值，表示结果。
  • has(val); // 判断该值是否存在于Set中。
  • clear(); // 清楚所有的值

set的遍历方法

set的键和值是同一个值。

  • keys() 返回键名的方法
  • values() 返回键值的方法
  • entries() 返回键值对的方法
  • forEach() 用回调函数遍历各个元素的方法
